This folder contains documentation generated from the ontology files located in 
/won-vocab/src/main/resouces/ontology

The HTML docs were created using the OWLDoc Protege plugin.
The ps file was created using html2ps:
> html2ps -W r -o message-owldoc.ps message/index-all.html

The PDF docs were created from ps using ps2pdf:
> ps2pdf message-owldoc.ps message-owldoc.pdf
