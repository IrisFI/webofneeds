Claimc back to ToC or Class ToC

IRI: https://w3id.org/won/agreement#Claim

A Message that claims other Messages using `msg:claims` is a Claim. It can become an Agreement via `msg:accepts` or it can be rejected via `msg:rejects`. The difference between a Claim and a Proposal is that the claim's content is interpreted as agreed-upon until rejected, while a Proposal's content is only interpreted as agreed-upon after it is accepted.

Proposalc back to ToC or Class ToC

IRI: https://w3id.org/won/agreement#Proposal

The Message that proposes other Messages or proposes to cancel Agreements is a Proposal. The difference between a Claim and a Proposal is that the claim's content is interpreted as agreed-upon until rejected, while a Proposal's content is only interpreted as agreed-upon after it is accepted.
