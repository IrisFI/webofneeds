IRI: https://w3id.org/won/agreement#Agreement

The message that accepts a proposal concludes an agreement. It is the last message that is taken into account when calculating the content of the agreement, which makes it suitable for identifying the agreement. Therefore, it is given the type Agreement.

acceptsop back to ToC or Object Property ToC

IRI: https://w3id.org/won/agreement#accepts

Used to link two message resources. The message in the subject accepts the message in the object. This only makes sense if the object references a proposal. During the projection from the edited conversation dataset to the agreement dataset, the validity of agreements is checked. Proposal messages that were not accepted are dropped, as are invalid or cancelled Agreements.
