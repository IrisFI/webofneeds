Object Properties

  • retracts

retractsop back to ToC or Object Property ToC

IRI: https://w3id.org/won/modification#retracts

Used for message editing. The subject and object must both be a Message. The meaning of the statement is that the subject message causes retraction of the object message. Retracting a message has the effect that in the projection from the acknowledged conversation dataset to the modified conversation dataset, all content graphs of that message are omitted.
