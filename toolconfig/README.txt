EXPORTING:
To export IntelliJ settings, go to 
  File->Export Settings... 
and select these options:
* Code Folding Settings, ...
* Code style schemes, ...
* Default project settings

select the jar file in this folder and overwrite it. Then delete the unpacked folder (from previous version) and unpack the new jar file.


IMPORTING: 
To import settings, go to 
  File->Import Settings...
and select the settings you like
